# Screenshots

This folder contains screenshots for the project README.

## Required Screenshots

Add the following screenshots to this folder:

1. **dashboard_home.png** - Home page showing the 4 main action tiles
2. **analyze_task.png** - Task analysis interface with form
3. **audit_trail.png** - Audit log viewer showing decision history  
4. **agent_insights.png** - Analytics dashboard with charts

## How to Take Screenshots

1. Start the system: `make start`
2. Login with password: `demo123`
3. Navigate to each page and take screenshot
4. Save with the exact filenames above

## Image Specifications

- Format: PNG
- Recommended width: 1200-1600px
- Compress for web (keep under 500KB each)
